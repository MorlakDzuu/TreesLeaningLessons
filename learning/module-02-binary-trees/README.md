# Модуль 2: Бинарные деревья и базовые операции

## Цель модуля
Освоить работу с бинарными деревьями и их основные операции.

## Задачи модуля
- [ ] Реализовать BST с операциями Insert/Search/Delete
- [ ] Реализовать все типы обходов (Pre-order, In-order, Post-order, Level-order)
- [ ] Реализовать обход Морриса
- [ ] Найти k-й наименьший элемент
- [ ] Проверить, является ли дерево валидным BST

## Теория
Бинарные деревья поиска, обходы деревьев, продвинутые операции.

## Практика
Реализовать все операции на бинарных деревьях поиска.
