# Модуль 0: Подготовка и настройка окружения

## Цель модуля
Подготовить инструменты и окружение для эффективного обучения.

## Задачи модуля
- [ ] Установить .NET SDK (последняя LTS версия)
- [ ] Настроить IDE (Visual Studio / Rider / VS Code)
- [ ] Установить расширения для визуализации (опционально)
- [ ] Создать репозиторий для практических заданий
- [ ] Настроить unit-тестирование (xUnit/NUnit)
- [ ] Установить инструменты для бенчмарков (BenchmarkDotNet)

## Теория
Этот модуль посвящён настройке рабочего окружения для изучения структур данных на деревьях.

## Практика
Создайте базовый проект с тестовой структурой и убедитесь, что всё работает корректно.
