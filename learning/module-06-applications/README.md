# Модуль 6: Применение в реальных системах

## Цель модуля
Научиться применять деревья в реальных проектах.

## Задачи модуля
- [ ] Изучить индексы в базах данных
- [ ] Изучить Expression Trees в .NET
- [ ] Реализовать парсер арифметических выражений
- [ ] Изучить Decision Trees в ML

## Теория
B+-деревья в БД, AST, Expression Trees, Decision Trees.

## Практика
Оптимизировать запросы в БД, построить AST для выражений.
