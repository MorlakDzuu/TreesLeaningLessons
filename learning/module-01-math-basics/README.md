# Модуль 1: Математические основы и анализ сложности

## Цель модуля
Закрепить формальные основы для анализа структур данных.

## Задачи модуля
- [ ] Изучить основы теории графов
- [ ] Понять анализ сложности алгоритмов
- [ ] Изучить математические доказательства для деревьев
- [ ] Реализовать функции вычисления высоты и подсчёта листьев

## Теория
Основы теории графов, анализ сложности, математические доказательства свойств деревьев.

## Практика
Реализовать функции для работы с деревьями и проанализировать их сложность.
