# Модуль 3: Самобалансирующиеся деревья

## Цель модуля
Изучить и реализовать самобалансирующиеся структуры данных.

## Задачи модуля
- [ ] Реализовать AVL-дерево с нуля
- [ ] Изучить красно-чёрные деревья
- [ ] Сравнить AVL и RB-tree
- [ ] Протестировать на больших данных

## Теория
AVL-деревья, красно-чёрные деревья, операции балансировки.

## Практика
Реализовать самобалансирующиеся деревья и сравнить их производительность.
